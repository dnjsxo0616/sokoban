# 실행 결과

- 입력 명령
  - w: 위쪽
  - a: 왼쪽
  - s: 아래쪽
  - d: 오른쪽
  - q: 프로그램 종료

1. 입력 명령에 따라 플레이어 'P'의 위치를 이동시키고 프로그램을 종료한다.


2. 이동 중 벽이나 공등 다른 물체에 부딪히면 (경고!) 해당 명령을 수행할 수 없습니다! 라는 메시지를 출력하고 플레이어를 움직이지 않는다.


3. 지원하지 않는 명령을 입력했을 경우 (경고) 지원하지 않는 명령입니다! 라는 메시지를 출력하고, 다시 사용자의 입력을 받는다.
